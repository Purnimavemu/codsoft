# Face Detection using MediaPipe

## 📌 Project Description

This project detects human faces in images using MediaPipe's pre-trained face detection model.

## 🚀 Technologies Used

* Python
* OpenCV
* MediaPipe

## 📂 How to Run

1. Install dependencies:
   pip install mediapipe opencv-python

2. Run the code:
   python main.py

## 🖼 Output

Displays image with detected face bounding box.
